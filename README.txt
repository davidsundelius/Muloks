==MULOKS==
Muloks is a clone of the classic Dreamcast game Chu chu rocket.
It is designed to be a multiplayer game for up to 8 people over
LAN. The game is written in Java and OpenGL (pre v2) and uses JOGL as a wrapper.

Muloks is released under the GNU GPL v2 licence (for more information see LICENCE.txt)
